## What I'm building

- **[tokenman](https://github.com/verkyyi/tokenman)** — a thin control layer for Claude inside GitHub Actions. Explicit trust boundaries via `read_paths` / `write_paths`, self-healing pipeline, feedback learning loop.
- **[agentfolio](https://github.com/verkyyi/agentfolio)** — open-source agentic portfolio engine. Fork, drop in your resume, deploy a site that adapts to each target role via Claude. [Live demo →](https://lianghuiyi.com/)
- **[github-agent-runner](https://github.com/verkyyi/github-agent-runner)** — Claude Code plugin for conversational discovery and installation of GitHub agentic workflows (`gh-aw`) with OAuth subscription setup.
- **[always-on-claude](https://github.com/verkyyi/always-on-claude)** — always-on Claude Code dev environment on EC2 + Docker Compose (~$22/mo). BYO auth.
